## 1 、什么是余币宝

**对于理财用户：**

余币宝有助于将有闲置资产和有借款需求的用户相匹配，用户申购余币宝后，每个整点将由系统根据用户设置的借出利率和实际借款需求判定是否借出成功以及该整点的利率，借出成功可以获得该小时的利息 **。**

**对于借贷用户：**

根据市场需求，提供更新符合流动性限制的利率的借贷服务。

**项目价值：**

1.  通过将资金来源对接至余币宝，释放垫资账户，从而降低平台风险。

2.  新的借贷机制能更好地调控和管理利率水平。

3.  额外奖励功能可以提供拉新促活和币种推广的功能

4.  利息复利有助于积累理财池资产

5.  提供活期,定期产品,支持灵活存取,无锁定期限制,定时赎回机制,资金流动性接近活期存款

6.  多币种支持,覆盖主流币种（BTC/ETH/USDT等）及部分山寨币
[图片上传中...(image.png-d29eee-1753490652146-0)]

7.  低门槛最低1 USDT即可参与 ,无复杂操作流程

8.  根据市场动态利率调整供需实时调节APY 历史年化3%-15%浮动收益

9.  自动复投收益每日自动结转 ,支持本息自动复投模式

## 历史记录

首页-用户数据下方可点击历史记录，进入历史记录弹窗页面，每页显示6条数据，只有利息收益的日期到日，其他都到时分秒

![image.png](https://upload-images.jianshu.io/upload_images/12041469-e6e6f0c2da277e43.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240)


相关php接口说明：

获取历史记录列表：/uni-loan/earn/history，返回时间戳格式的历史记录

获取历史记录类型：/uni-loan/earn/history/types

*   利息收益type=3：按币种，统计用户借出成功的利息收益

对应数据库表：loan_lend_earn_interest_daily_record 按日统计的理财利息收益表，查询sql：

```
select * from uni_loan.loan_lend_earn_interest_daily_record where user_id='6789'
    and asset='USDT' and create_time >= '2024-04-01 00:00:00' and create_time <= '2024-04-01 23:59:59'
             order by id desc;
```

明细数据查loan_lend_earn_interest_record 发放利息记录表

*   申购type=1：

对应数据库表：loan_lend_record 用户借出、赎回记录表，查询sql：

```
select *
from uni_loan.loan_lend_record
where user_id = '6789'
  and asset = 'BAT'
  and type in (1,5)
  and status = 1
  and create_time >= '2024-04-11 00:00:00'
  and create_time <= '2024-04-11 23:59:59'
order by id desc;
```

*   赎回type=2：

对应数据库表：loan_lend_record 用户借出、赎回记录表，查询sql：

```
select * from uni_loan.loan_lend_record where user_id='6789'
        and asset = 'BAT' and type=2 and status=1
            and create_time >= '2024-04-11 00:00:00' and create_time <= '2024-04-11 23:59:59'
                                    order by id desc;
```
